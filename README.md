# Modern Personal Website

A visually stunning, single-page personal website built with a modern aesthetic, featuring glassmorphism elements, dynamic animations, and a live clock.

## Features Implemented

*   **Core Structure**: Clean and semantic HTML5 layout (`index.html`).
*   **Modern Typography**: Utilizes Google Fonts with **Outfit** for elegant headings and **Inter** for readable body text.
*   **Dynamic Background**: Features a continuously animating linear gradient background with deep purple and indigo tones.
*   **Visual Depth**: Three floating, blurred orbs provide an immersive sense of depth behind the main content.
*   **Glassmorphism Card**: The central element is a premium glass card achieved using CSS `backdrop-filter: blur(20px)` and semi-transparent backgrounds to create a frosted-glass effect.
*   **Interactive Animations**:
    *   **Shimmer Effect**: A beautiful purple-to-white shimmer gradient animation applied to the name.
    *   **Entrance**: Staggered fade-up animations reveal elements sequentially upon page load.
    *   **Hover Lift**: A sophisticated lift-and-glow effect when hovering over the main card.
*   **Live Functionality**: A fully functional live clock and formatted date display, parsing and updating the precise local time every second (`script.js`).
*   **Responsive Design**: The layout adapts seamlessly across desktop, tablet, and mobile devices.

## Setup

Simply open `index.html` in your web browser to view the page. No build process or server is required.
